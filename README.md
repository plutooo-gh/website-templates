## 🚀 Usage

```bash
# 1. Clone the repository
git clone https://github.com/Plutooo/website-templates.git

# 2. Navigate into a template
cd website-templates/template-01

# 3. Open in browser
open index.html
```

No build step. No install. Just open `index.html` and go.
